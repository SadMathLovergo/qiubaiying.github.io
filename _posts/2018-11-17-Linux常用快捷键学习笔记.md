## Ubuntu终端快捷键

| 功能 | 快捷键 |
| ---- | ---- |
| 打开终端 | Ctrl+Alt+T |
| 新建终端窗口 | Ctrl+Shift+N |
| 关闭终端窗口 | Ctrl+Shift+Q |
